📸 Image Gallery App
📖 Description

This is a responsive Image Gallery Web Application built using React.js. The application allows users to browse a collection of images with features like search, category filtering, and interactive UI effects. It is designed with a clean and modern interface and works smoothly on both desktop and mobile devices.

🚀 Features
🔍 Search images by title
🎯 Filter images by categories (Nature, Beach, Desert, Animals)
📱 Responsive design with mobile sidebar menu
🎨 Hover effects with image overlay (title & description)
🧩 Component-based structure (Navbar, Gallery, ImageCard, Footer)
💡 Clean and user-friendly UI
🛠️ Technologies Used
React.js
JavaScript (ES6+)
HTML5
CSS3 (Flexbox & Grid)
